参考式链接

1
2

参考式链接 = [内容][参考标识符]
参考标识符 = [标识符]: 地址 "标题"

参考式链接 和 参考标识符 的定义没有先后顺序；

[内容] 和 [参考标识符] 之间可以有一个空格，也可以没有空格；

如果 内容 和 参考标识符 一样，也可简写成 [参考标识符] [] ;

标题 是可选的，可以用单引号、双引号或是圆括弧包着；

锚点就是一个链接

标签形式的锚点目标

因为MarkDown链接会被转成a标签，并且MarkDown中也可以写标签，所以可以利用HTML的锚点机制直接定义一个带 id 特性的任意标签作为锚点目标，然后把MarkDown中的锚点地址的目标内容设置为 id 特性的值；这样便可以实现页面内跳转；

1

<标签名 id="目标内容">元素内容</标签名>
  • 标签形式的锚点目标的id特性值中是不能含有中文字符；
1

<div id="这是锚点目标">跳转到这里</div>
