flow

概述

流程图的语法大致分为两部分。

  • 定义元素
  • 连接元素

定义元素

语法概览

tag=>type: content:>url

语法解释

  1. tag：标签，用于连接元素时使用
  2. type：该标签的类型。共有6种类型如下：
标签类型 解释
start 开始节点
end 结束节点
operation 操作节点
subroutine 子程序节点
condition 条件节点
inputoutput 输入或产出节点

  1. content：流程语句中放置的内容

    • content之前要有一个空格
    • 可插入|将之后的内容作为备注（此项的真实目的待确定）

  2. url：链接，与流程语句绑定

    • 需要注明协议类型，已知可使用的：
      • http://www.qq.com
      • ftp://127.0.0.1
      • d:\download
      • E:\11\111.png
      • file:///E:\11\111.png

连接元素

使用->符号，如：

c2(yes)->io->e
c2(no)->op2->e
  1. condition 节点将会有 yes 和 no 两个分支
    • co(yes)
  2. 所有元素节点可以控制方向
    • op(right)
    • 方向默认为向下，可选（left,right,bottom,top）
  3. condition 节点分支与方向同时存在时
    • co(yes,right)

案例

#语法
st=>start: 开始|rem:>ftp://127.0.0.1
e=>end: 结束|rem:>d:\download
op=>operation: 操作|rem:>E:\11\111.png
su=>subroutine: 子程序|rem:>file:///E:\11\111.png
co=>condition: 条件|rem:>https://www.qq.com
in=>inputoutput: 输入或产出|rem:>https://www.qq.com
st(right)->op(bottom)->su->co(yes,right)->in(bottom)->e
co(no)->e
in(top)->op
